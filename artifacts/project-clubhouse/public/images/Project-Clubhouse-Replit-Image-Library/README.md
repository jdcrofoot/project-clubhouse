# Project Clubhouse Image Library for Replit

This package contains 70 individually cropped Project Clubhouse concept images.

## Folder structure

- `01-aspirational/` — the original future-state visual library. Use sparingly for long-term vision, brand storytelling and future flagship concepts.
  - `01-architecture/` — facility and space concepts
  - `02-community/` — fitness, mentorship, studying and belonging
  - `03-website-heroes/` — images composed for headline overlays
  - `04-app-pulse/` — app and Pulse Board integration concepts
- `02-pilot/` — the recommended visual library for the first Prior Lake location.
  - `01-core/` — workout floor, sportsbook room, half court and entry
  - `02-study-fuel/` — shared study, individual pods, group study and fuel station
  - `03-outdoor/` — patio, basketball and pickleball concepts
  - `04-workout-pulse/` — workout-floor views with leaderboards and tickers
  - `05-clubhouse-study/` — clubhouse, shared study and individual pods
  - `06-fuel-connections/` — fuel station and connected-space overviews
  - `07-sportsbook/` — sportsbook-inspired clubhouse in multiple operating modes

## Replit implementation guidance

Default to images under `02-pilot/` for the current website. Treat `01-aspirational/` as future-state material and do not mix its premium architectural scale into pages describing the first location.

Recommended placements:

- Homepage hero: `02-pilot/01-core/pc-pilot-core-01.png` or `02-pilot/07-sportsbook/pc-pilot-sportsbook-a-01.png`
- Workout experience: `02-pilot/04-workout-pulse/`
- Clubhouse/sportsbook: `02-pilot/07-sportsbook/`
- Study and mentorship: `02-pilot/02-study-fuel/` and `02-pilot/05-clubhouse-study/`
- Fuel station: `02-pilot/06-fuel-connections/`
- Outdoor spaces: `02-pilot/03-outdoor/`
- Long-term vision page: `01-aspirational/01-architecture/`

## Brand lock

Keep the black industrial shell, charcoal finishes, economical plywood/butcher-block details and restrained neon-lime accents. The pilot should feel energetic and desirable but visibly attainable: an adaptive-reuse neighborhood clubhouse, not a luxury athletic club.

Avoid language or design treatments associated with gambling. “Sportsbook-inspired” refers only to comfortable tiered viewing, multiple sports screens and shared game-day energy.
